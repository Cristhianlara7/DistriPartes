Para la ejecución del sistema de inventarios puedes seguir los siguientes pasos:

1. Descarga el repositorio en una ruta conocida y de fácil acceso.
2. Crea una base de datos con la estructura que se tiene en el archivo pdo.sql que se encuentra dentro de la carpeta DB o importa este archivo en un gestor como PHPMyAdmin.
3. Debes contar con Laragon o Xampp para iniciar los servicios MySQL y Apache.
4. Copia la carpeta del repositorio dentro de la carpeta www de Laragon o Xampp la cual debe estar dentro del disco c *** ejemplo: C:\laragon\www 
5. De acuerdo con la configuración establecida, puedes ejecutar el proyecto desde localhost/"nombre del proyecto" ejemplo: http://localhost/Distripartes/"
6. El usuario inicial para ingresar es: Administrador y la contraseña: Administrador 
A partir de ahí, puedes empezar a crear usuarios.
